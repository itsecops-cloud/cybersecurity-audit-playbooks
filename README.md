# Cybersecurity Audit Playbooks

This repository contains audit-ready cybersecurity playbooks, control mappings, and operational guidance for organizations preparing for:

- SOC 2 Type I & Type II
- ISO/IEC 27001
- CMMC Level 2 (NIST SP 800-171)
- General cybersecurity audits and assessments

The content is written from an IT operations and audit-readiness perspective, focusing on real-world enforcement, evidence collection, and assessor expectations.
