# Cybersecurity Audit Playbooks

This repository contains audit-ready cybersecurity and compliance playbooks
focused on SOC 2 Type II, ISO/IEC 27001, and CMMC Level 2 (NIST SP 800-171).

These documents are written from an IT operations and audit-readiness perspective
and are intended for CISOs, compliance managers, auditors, and IT security teams.
